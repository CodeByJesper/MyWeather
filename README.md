# MyWeatherApp

A simple weather application that automatically detects your location and shows the current weather using the OpenWeather API.

## Status

This project is still a work in progress.  
most features are not fully implemented and may change.

## Features

- Automatically gets your current location using the browser's Geolocation API
- Fetches current weather data from OpenWeather
